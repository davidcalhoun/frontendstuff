## Code
```html
#example {
  width: 100px;
  height: 100px;
  background-color: red;
     -moz-border-radius: 12px; /* FF3-4 */
  -webkit-border-radius: 12px; /* Saf3-4 */
          border-radius: 12px; /* Opera 10.5, IE 9, Saf5, Chrome */
}
</style>

<div id="example">
</div>
```
